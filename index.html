How to!

It is extremely simple to use, you need to create a div around every single paragraph, and a element after this (to prepend the image you want)

If you set a class to the optional params arrowDownClass & arrowUpClass, they will be used to reset the class on mouseover and mouse leave, giving the animation you can see at the examples (top of the page) courtesy of fontawesome



So, the HTML structure is as it follows:

<div class="some-class"> <!-- or id -->
    <p>the paragraph</p>
</div>
<div></div> <!-- image will be attached here -->

Usage:

$('.paragraph-container').hideParagraph({
    speed: 300, // optional
    arrow: '',
    arrowDownClass: 'fa-chevron-down', // optional
    arrowUpClass: 'fa-chevron-up', // optional
    paragLength: 100, // optional
    paragFixed: 55 // optional
});
